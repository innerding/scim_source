# SCIM Projektquelle - strukturierte Dokumentenbasis

Dieses Paket ordnet die bereitgestellten SCIM-Projektquellen in eine klare Dokumentenstruktur ein.

Einstiegspunkt:

```txt
docs/scim/.index.md
```

Die Originalinhalte wurden nicht inhaltlich umgeschrieben, sondern unter stabileren Zielnamen einsortiert. Neue ergänzende Dateien sind:

- `docs/scim/.index.md`
- `docs/scim/KI_ARBEITSANWEISUNG.md`
- `docs/scim/DATEI_MANIFEST.md`
- `.gitkeep`-Platzhalter in noch leeren Zielordnern
